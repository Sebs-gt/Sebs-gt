## 🧠 Cursos que más he disfrutado

Estos son algunos de los cursos que más me han marcado durante la carrera:

- 🖥️ **Organización Computacional**
  - Comprender cómo funciona un computador desde su arquitectura interna fue una experiencia increíble.
  - Me permitió entender la relación entre hardware y software, y cómo interactúan ambos para ejecutar programas.

- ⚙️ **Arquitectura de Computadores y Ensambladores I**
  - Uno de mis cursos favoritos.
  - Disfruté mucho trabajar directamente con componentes de hardware y programarlos utilizando lenguaje ensamblador.
  - Fue fascinante ver cómo el código puede controlar dispositivos físicos y comprender el funcionamiento interno de un procesador.


Siempre estoy abierto a aprender, colaborar y compartir conocimientos.
